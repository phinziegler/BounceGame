Bounce Game

I'm really not sure exactly what this is yet. So far it is just a fun experiment, but eventually I intend to turn this into an actual game, with actual objectives, AI, and maybe other game modes?

Controls

Red uses WASD to move, blue uses arrow keys.

Hold the jump key to go higher.

The player turns darker when they cannot jump.
